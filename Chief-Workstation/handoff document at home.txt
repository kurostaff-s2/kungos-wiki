 1. 前言                                                                                                                                                                                
     在上一篇文章中，我们详细讲解了如何配置一个基本的 Spring Security 项目，并实现了用户的登录和退出功能。                                                                              
     但是，如果用户没有登录就访问一个需要登录的页面，那么 Spring Security 会直接返回一个 403 错误，这显然不太友好。                                                                     
     所以，在本篇文章中，我们将实现一个自定义的登录页面，并在用户未登录时自动跳转到该页面。                                                                                             
 2. 项目环境                                                                                                                                                                            
     Spring Boot 2.7.7                                                                                                                                                                  
    Spring Security 5.7.4

 3. 实现步骤                                                                                                                                                                            
     3.1. 创建登录页面

  3.2. 配置登录页面                                                                                                                                                                     
  接下来，我们需要配置 Spring Security，使其在用户未登录时自动跳转到登录页面。                                                                                                          
  在 SecurityConfig.java 文件中，添加以下代码：                                                                                                                                         
  @Override

             .anyRequest().authenticated()  // 其他请求需要认证                                                                                                                         
             .and()                                                                                                                                                                     
         .formLogin()                                                                                                                                                                   
             .loginPage("/login")  // 指定登录页面                                                                                                                                      
             .permitAll();  // 允许所有用户访问登录页面                                                                                                                                 
 }
